# 키오스크 과제 소개


## 커밋 컨벤션
아래와 같은 형식을 따라 커밋하세요.

```
<b><타입>: <제목>

<본문>

<꼬리말><b>
```

### 1. 커밋 타입
- 🆕: 새 기능 추가
- 🔧: 기능 수정 & 개선
- 🐛: 버그 수정
- 📝: 문서 수정
- 🎨: 코드 스타일 변경(들여쓰기 등)
- 🏭: 코드 리팩토링(기능 변경 없이 구조만 개선한 경우)
- 📁: 파일 구조 변경

### 2. 제목
이 커밋에 대한 간략한 설명

예) Main 클래스 리팩토링

### 3. 본문(선택 사항)
이 커밋에 대한 추가 설명

### 4. 푸터(선택 사항)
기타 참고사항 기술
